Siemens PLM Software

Strategic Student Co-Op

As a Strategic Student Co-Op at Siemens PLM Software, I completed multiple tasks over a course of 4 months. My first project I was assigned to was one in which I was responsible for converting widgets used in their system, Teamcenter, from Java to Javascript. I did this by analyzing the functions of each widget and replicating them using Javascript. Within this project, I was responsible for creating icons used within the widgets. I performed this task by creating SVGs using Adobe Illustrator. This led me into my next task - to create SVGs icons for all of Teamcenter. Once I completed creating over 100 icons, I was required to create test cases using Cucumber and Selenium. These tests made sure that each function of Teamcenter was functioning properly before the next version was released.
