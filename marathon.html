Marathon Petroleum LP

IT Services Intern

As an IT Services Intern in the Enterprise Data Management Department at Marathon Petroleum, I was responsible for various tasks. Amongst my responsibilities was to maintain data integrity within the SAP Business Intelligence Web Reports created by various departments. I did this by validating all database connections were valid and all variables used were pointing to the correct tables within their respected SQL databases. Another responsibility was to create dashboards to track various metrics for different managers within the company. I did this by using SAP Business Intelligence Web Intelligence Reports and Information Design Tool. These tools allowed me to create universes that connected SQL data to a responsible web display of data. While working at Marathon Petroleum LP, I was also expected to complete migration requests by moving different reports into different testing environments. Once the reports were moved to our production environment, I was required to write various tests cases using HP Quality Center to validate that each report was running and presenting the correct information.
