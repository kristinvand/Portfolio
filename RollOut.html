Microsoft

Explorer Intern

As an Explorer Intern on the SPACE (Security, Privacy, Assurance, and Compliance Engineering) Team within the Artifical Intelligence Core Team at Microsoft I was responsible for maintaining integrity within the AI Core space. These responsibilities included writing C# scripts to detect vulnerabilities within the Microsoft Intranet. These scripts used active directory to find all users on the Microsoft network then located file shares that these users owned. Using regular expression matching, the scripts would detect any exposed credentials within these files. It would then notify these owners that there were vulnerabilities within their files that needed to be fixed.
